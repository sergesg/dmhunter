微信接入

请按以下指引接入 DM Hunter。

  1. 在微信公众平台 https://mp.weixin.qq.com/ 申请订阅号。主体为个人或法人均可，无需认证。

  2. 左侧菜单栏转到“开发 -> 基本配置”，修改服务器配置。

    • {% if url %}{{ url }}{% else %} 等待获取{% endif %}
    • {% if token %}{{ token }}{% else %} 等待获取{% endif %}严禁泄露
    • {% if aeskey %}{{ aeskey }}{% else %} 等待获取{% endif %}严禁泄露
    • 安全模式

    只要能成功保存就说明以上信息填写正确。

  3. 点击“启用”；从启用到生效可能有 5 分钟延迟。

    启用后，订阅号原本的自动回复会失效。向该订阅号发送 DM内容 测试，如有回复 弹幕发射~升空 则说明接入成功。

  4. 将以下 Token 填在弹幕客户端中。

    • {% if id and client_token %}{{id}}:{{ client_token }}{% else %} 等待获取{% endif %}持有该 Token 的任何人会获得该公众号收到的所有消息以便弹幕显示
{% csrf_token %}

注意事项

  1. 以上所有 Token、URL、密钥只会显示一次。如果遗忘了任何一项，只能重新接入（重做第 2 ~ 4 步）。
  2. 严禁泄露微信公众平台服务器配置的 Token 和密钥。
  3. 持有弹幕客户端 Token 的任何人会获得该公众号收到的所有消息以便弹幕显示。
